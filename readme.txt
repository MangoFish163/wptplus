=== WPTPLUS Basic ===

Requires at least: 6.2
Requires PHP: 7.4
WPTPLUS 是多个功能集合体的一个插件

== Description ==

WPTPLUS 是为了解决日常工作中各种需求的一个功能集合体，主要围绕着WooCommerce等进行一些插件开发
主要功能如下：

| 功能 | 简介 | 
| ------ | ------ |
| 来源跟踪 | 对用户注册和创建订单进行Referer和UTM进行记录，并在后台展示 |
| 推送飞书 | 当有用户创建订单时，推送订单信息到相应飞书群 |
| 多语言选择器 | 替代Gtranslate的原始语言选择器,并添加一个可选的弹出层 |

== Installation ==

1. 上传 `wptplus`目录 到 `/wp-content/plugins/` 目录
2. 激活插件，开始设置使用。
